Fast fused multiply and accumulate. Returns x * y + z.

A fused multiply and accumulate avoids the rounding that would normally happen after multiplication. This offers greater precision. Some microprocessors do not have an FMA instruction so ::std::fma() will be significantly slower than (x*y+z). Most DSP cases prefer speed over precision which this function ensures by using (x*y+z) on processors without an FMA instruction. If you require precision over speed, use ::std::fma() instead.
