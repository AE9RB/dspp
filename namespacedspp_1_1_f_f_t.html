Detailed Description

Fast Fourier/Cosine/Sine Transforms.

Performs a transform on arrays with power-of-two sizes. Uses Sande-Tukey decimation in frequency algorithm.

Example usage: 

std::vector<std::complex<float>> data(512);
dspp::FFT::dft(data);

C-style arrays are supported: 

std::complex<float> data[512];
dspp::FFT::dft(data, 1); // 1 = inverse DFT

Out-of-place transform: 

std::array<double,64> in;
std::array<std::complex<double>,32> out;
dspp::FFT::dft(in, out); // real transform

Planning is automatically performed as necessary. The first transform of a specific size and type will take longer than subsequent transforms.

Each of the transform types has an unsafe version. If possible, always use the convenience overloads which engage the type safety of your compiler.
